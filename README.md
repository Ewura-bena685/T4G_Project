# FRAU ROT — Menstrual Health Companion

**FRAU ROT** is a polished, privacy-conscious menstrual wellness web app designed to help users better understand their cycles, moods, symptoms, and daily health patterns.

Built as a static front-end project and hosted on **GitHub Pages**, the platform combines cycle tracking, wellness education, and a supportive user experience in a clean, modern interface.

---

## 🌐 Live Demo

**Website:** https://ewura-bena685.github.io/T4G_Project/

---

## ✨ Project Overview

FRAU ROT was created as a digital menstrual wellness companion that encourages cycle awareness through thoughtful design and easy-to-use tools. The app helps users log period data, monitor cycle rhythms, explore wellness content, and gain simple insights from their patterns.

The experience is designed to feel:
- **warm and supportive**
- **clear and empowering**
- **modern and accessible**
- **private by default**

---

## 🚀 Core Features

### User Experience
- Elegant landing page with product introduction
- Responsive navigation and polished UI styling
- Sign up and sign in flows
- Mobile-friendly layout for desktop, tablet, and phone

### Menstrual Wellness Tools
- Cycle length setup and personalization
- Period logging and daily tracking
- Dashboard summaries and phase-based insights
- Cycle-aware educational content
- Learn and blog sections focused on menstrual wellness

### Support & Awareness
- Partner sharing/invite flow
- FAQs and contact page
- Evidence-informed wellness articles
- Simple guidance around moods, symptoms, and cycle phases

### Privacy
- User state is stored locally in the browser using `localStorage`
- No backend or external database is required for the demo

## 🧭 How the App Works

1. **Start on the homepage** to explore FRAU ROT.
2. **Create an account** or sign in.
3. **Set your cycle length** and log period information.
4. **View your dashboard** for daily summaries and predictions.
5. **Explore the Learn/Blog pages** for wellness guidance and cycle education.

## 🎨 Design Direction

The visual identity of FRAU ROT is inspired by modern health-tech products and feminine wellness platforms. The UI uses:

- soft rose and blush tones
- rich burgundy accents
- clean editorial typography
- glassmorphism-inspired surfaces
- premium shadows, spacing, and rounded components

This creates a product experience that feels both **professional** and **reassuring**.
